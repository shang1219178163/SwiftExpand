模块速览

按场景浏览常用 API。输入关键词可即时筛选。

RT
Foundation

Runtime 与模型

遍历 ivar / 属性 / 方法，模型自动编解码，字典与模型互转。

let model = User(dic: jsonDict) model.se_encode(with: coder) "hello".attributed.font(.boldSystemFont(ofSize: 16)).color(.red)
UI
UIKit

视图与手势

闭包式手势注册，UIControl 事件转回调，减少 target-action 样板。

view.addGestureTap { reco in DDLog(reco.view) } button.addActionHandler { sender in sender.isSelected.toggle() }
CV
列表

Table & Collection

类型安全的 dequeue，Header/Footer 与 Cell 复用一行搞定。

let cell = MyCell.dequeueReusableCell(tableView) let view = MyHeader.dequeueReusableHeaderFooterView(tableView)
DT
时间

DateFormatter 缓存

线程内复用 Formatter，日期字符串与时间戳互转。

DateFormatter.stringFromDate(date) DateFormatter.intervalFromDateStr("2024-01-01") DateFormatter.stringFromInterval(timestamp)
RX
反射

类名反射

ObjC / Swift 类名解析，字符串创建 ViewController。

if let cls = NNClassFromString("HomeVC") { } let vc = UICtrFromString("DetailViewController")
+
跨平台

网络与 macOS

URL / URLSession 辅助，AppKit 窗口、菜单、表格等扩展。
